## 🎓 How to Use This Project

### For Developers
1. Clone the repository
2. Run `npm install`
3. Start with `npm run dev`
4. Make changes to pages, components, or data
5. Build with `npm run build`
6. Deploy to Cloudflare Pages or GitHub Pages

### For Content Managers
1. **Add Companies**: Edit `src/data/companies.ts`
2. **Add Blog Posts**: Edit `src/data/blog.ts`
3. **Update Team**: Edit `src/data/leadership.ts`
4. **Update FAQs**: Edit `src/data/faqs.ts`
5. **Update Timeline**: Edit `src/data/timeline.ts`

### For Designers
1. **Colors**: Edit CSS variables in `src/styles/global.css`
2. **Typography**: Update font links in `src/layouts/BaseLayout.astro`
3. **Components**: Modify Astro components in `src/components/`
4. **Pages**: Edit layouts and styling in page files
